小聚

  • June 16, 2012
  • Lynx

夜里坐在回程的大巴上。 车窗上蒙着厚重的雾气 ，只见得远处发散的光点和昏暗车厢内座椅的倒影。想和舍友们好好地道别，却说不出什么话来……

来时，沿途的风光尽是熟悉与亲切。这个季节校道上开满了大花紫薇和洋紫荆。对于“校园”始终是我一辈子不想离开的地方。想起来去年的6月，最难过的莫过于最后一次走出图书馆和宿舍大门的时候……

和舍友们的小聚，也不过3,4小时，没有什么特别想做的事，听听熟悉的声音，知道有些东西还未改变便也就好了。尧要会南平了，很难想象在将来的日子还能有几次见面。

她每次来时都是静静的雨夜

  • May 24, 2012
  • Lynx

夜里翻来覆去地睡不着，断断续续地看着未读完的书，终究是倦了，却不想睡觉。恍惚间点开了KF里的《国境以南，太阳以西》。我本是少有重读习惯的，更何况这个短小的故事自觉得记得还算真切。

书中的感情，记忆，以及当时的自己的感情，猛然间一股脑地膨胀开来。想起大二的日子，或许那才是自己真正的样子。或许人在阅读的过程中自然是在不断地汲取，但说不定也把什么落在了书里。

梦回大院

  • February 19, 2012
  • Lynx

离开大院已经有13年了。如今的家距离大院并不算远——约莫15分钟的路程。然而那里终究是回不去的地方。

前些日子，想着到和大院紧邻的芝山公园转转，或许可以从外围见见这童年的地方。

大院正门较之以前拓宽了有一倍多，偏门站着一个手里拿刺（和谐）dao的小兄弟，正门则有三个仿佛胸前绑着手（和谐）榴（和谐）弹的家伙来回走正步。。

绕进芝山公园的正门，是长长的上坡，这里与大院仅有一墙之隔，临墙栽着在其他地方少见的白千层。因为一年四季都在蜕皮显得苍老而不美观。想来该是三四十年前的道行树了吧。从google earth上看，大院内的房子大体都已经换了一轮。唯独小时候住过的房子竟不曾变过。

2012年1月12日

  • January 12, 2012
  • Lynx

今天突然难过起来，实在说不上什么缘由，大概还是想到自己的人生以及将来的生活。看不下闲书，若说什么都不想，好好地享受生活，却也是做不到的。快乐和忧伤都是，多半都是自己平添出来的心结。可我还是希望继续纠结着吧。

夜里都在听《蒋勋细说红楼梦》，多数时候都是开心的，只可惜竟找不到一个人能分享这份喜悦。看到宝玉和黛玉的感情，既羡慕又禁不住难过起来。想好好地和你聊一聊，却也已经做不到。
